# Инструкция по запуску проекта

Скачать архив проекта с Git и разархивировать. 
1 вариант
![image](https://github.com/Nikitagm28/Check-heros-dota2/assets/71442136/1e952552-7e60-4b7b-bfc0-26d0bf94a619)
